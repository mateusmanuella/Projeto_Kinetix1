# Sprint 1

## Objetivo da Sprint

Definir escopo do MVP, levantar requisitos, estruturar documentação inicial e desenhar a base conceitual do sistema.

## Backlog da Sprint

- Levantamento de requisitos funcionais e não funcionais
- Definição do MVP
- Identificação de perfis de usuário
- Mapeamento das jornadas principais
- Regras de negócio iniciais
- Início do backlog no Kanban
- Início dos casos de uso
- Início do DER
- Definição das entidades centrais
- Esboço da arquitetura geral

## Tarefas dos Desenvolvedores com Notebook

### Backend
- Definir arquitetura do backend
- Estruturar camadas e pacotes
- Definir entidades centrais
- Esboçar endpoints principais

### Frontend
- Iniciar estrutura do projeto React
- Definir rotas do painel do fisioterapeuta
- Planejar telas do MVP

### Mobile
- Iniciar estrutura do app Expo
- Definir navegação principal do app
- Planejar telas do paciente

### Banco
- Desenhar o DER inicial
- Definir tabelas centrais
- Identificar chaves e relacionamentos

## Tarefas dos Integrantes com Celular

### Figma
- Wireframes iniciais
- Fluxo visual das telas principais

### Documentação
- Introdução do TCC
- Justificativa
- Objetivos geral e específicos

### UML
- Diagrama de caso de uso inicial
- Diagrama de contexto

### Casos de Uso
- Descrição textual dos casos de uso principais

### SQL
- Levantamento preliminar de tipos de dados
- Estrutura conceitual das tabelas

### Testes
- Lista inicial de critérios de aceite
- Cenários básicos de validação

### LGPD
- Identificação de dados pessoais e sensíveis
- Levantamento de cuidados mínimos

### Slides
- Estrutura base da apresentação final

## Critérios de Aceite

- Escopo do MVP definido
- Backlog organizado
- DER inicial criado
- Casos de uso principais iniciados
- Protótipo inicial esboçado
- Estrutura documental aberta

## Principais Riscos

- Escopo inflado
- Discussões sobre funcionalidades avançadas
- Modelagem excessiva no início

## Mitigação

- Foco exclusivo no MVP
- Congelamento de funcionalidades não essenciais
- Validação constante com o grupo

## Entregável para Professores

- Visão geral do projeto
- Escopo do MVP
- Requisitos iniciais
- DER rascunho
- Casos de uso preliminares
- Protótipo inicial
