# Sprint 3

## Objetivo da Sprint

Iniciar o núcleo técnico do sistema com backend, banco, autenticação e base dos CRUDs.

## Backlog da Sprint

- Configuração do backend Spring Boot
- Configuração do SQL Server
- Flyway e primeira migration
- Implementação das entidades base
- Cadastro e login
- JWT
- Segurança básica
- Estrutura base do frontend
- Estrutura base do mobile
- Integração inicial com API

## Tarefas dos Desenvolvedores com Notebook

### Backend
- Criar projeto Spring Boot
- Configurar dependências
- Configurar conexão com o banco
- Criar migrations iniciais
- Implementar entidades de usuário e perfis
- Implementar login e registro
- Gerar JWT
- Configurar Spring Security
- Criar endpoints de autenticação
- Criar tratamento básico de exceções

### Frontend
- Criar layout base do login
- Configurar rotas
- Estruturar páginas principais
- Iniciar consumo da autenticação

### Mobile
- Criar projeto Expo
- Configurar navegação
- Criar tela de login
- Estruturar consumo inicial da API

### Banco
- Aplicar script inicial
- Validar relacionamentos
- Conferir constraints e índices

## Critérios de Aceite

- Backend rodando localmente
- Banco criado
- Login funcionando
- JWT emitido
- Base de frontend e mobile pronta

## Principais Riscos

- Ambiente atrasar
- Erros com SQL Server
- Falhas básicas no login

## Mitigação

- Configuração simples
- Teste de conexão cedo
- Validação do fluxo mínimo
