# Visão Geral

## Projeto

KINETIX

## Tema

HealthTech focada em prevenção, acompanhamento e reabilitação física.

## Objetivo

Criar uma plataforma para auxiliar fisioterapeutas e pacientes com login, gestão clínica, prescrição, dor, agendamento e evolução.

## Stack prevista

- Backend: Java 21, Spring Boot 3, Spring Security, JWT, Flyway, SQL Server
- Frontend: React, Vite, React Router, Axios, React Hook Form, Zod
- Mobile: React Native, Expo, Expo Router, AsyncStorage, TanStack Query
